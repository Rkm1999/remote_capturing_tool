# Remote Capture v0.1 Beta

## Scope

This first Android beta validates still-photography control over Sony's legacy
ScalarWebAPI camera service. Movie recording is intentionally excluded.

## Included

- Wi-Fi discovery, live view, physical and app shutter capture
- Single and continuous-shooting imports
- Exposure, drive, burst-speed, and compatible power-zoom controls
- Live ND, Live Composite, and Panorama processing
- Built-in LUT previews and local 3D `.cube` LUT import
- Standard/High live-view selection when supported
- Optional phone GPS coordinates in saved JPEG EXIF
- MediaStore output, Original import where advertised, history, and diagnostics

## Tested

- Samsung Galaxy S25 (`SM-S931W`) running Android 16
- Sony Alpha A6300 (`ILCE-6300`) using its legacy remote camera application

Other legacy ScalarWebAPI Sony cameras are capability-detected but have not yet
been verified. Controls may differ by model, firmware, installed camera app,
lens, and current shooting mode.

## Beta Notes

- The attached APK is debug-key signed for direct evaluation.
- Geotagging is disabled by default and uses the phone location only after the
  user grants precise foreground location permission.
- Imported LUT data is processed locally.
- RAW and movies remain on the camera card.
